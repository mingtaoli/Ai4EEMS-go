# 服务和端口说明

## 服务端口分配

- **19801 系列**: Go-Zero 后端服务端口
  - 账号服务（Account Service）: **19801**
  - 登录服务（Login Service）: **19802**
  - 权限管理服务（Permission Management Service）: **19803**
  - 设备信息服务（Device Information Service）: **19804**

- **29801 系列**: Julia 计算引擎服务端口
  - 优化计算服务（Optimization Calculation Service）: **29801**
  - 数据分析服务（Data Analysis Service）: **29802**

- **9801 系列**: 前端服务端口
  - 用户管理界面（User Management Interface）: **9801**
  - 系统管理界面（System Management Interface）: **9802**

仍然可能变化。以最新的设置为准。
