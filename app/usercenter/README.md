以下是以 `https://www.ai4energy.com/usercenter/` 为基础路径，列出的基本需要的路由：

1. **POST `https://www.ai4energy.com/usercenter/admin_register`**
   - 功能：管理员注册账号

2. **POST `https://www.ai4energy.com/usercenter/changePassword`**
   - 功能：用户修改密码

3. **POST `https://www.ai4energy.com/usercenter/setUserAuthority`**
   - 功能：设置用户权限

4. **DELETE `https://www.ai4energy.com/usercenter/deleteUser`**
   - 功能：删除用户

5. **PUT `https://www.ai4energy.com/usercenter/setUserInfo`**
   - 功能：设置用户信息

6. **PUT `https://www.ai4energy.com/usercenter/setSelfInfo`**
   - 功能：设置自身信息

7. **POST `https://www.ai4energy.com/usercenter/setUserAuthorities`**
   - 功能：设置用户权限组

8. **POST `https://www.ai4energy.com/usercenter/resetPassword`**
   - 功能：重置用户密码

9. **POST `https://www.ai4energy.com/usercenter/getUserList`**
   - 功能：分页获取用户列表

10. **GET `https://www.ai4energy.com/usercenter/getUserInfo`**
    - 功能：获取自身信息

11. **POST `https://www.ai4energy.com/usercenter/register`**
    - 功能：用户注册



这些路由映射到相应的功能，可以用于实现用户中心的基本操作。
